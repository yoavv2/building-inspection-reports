# Product Spec — Initial Mobile Scaffold

## מטרת הגרסה

הקמה של שלד Expo + TypeScript עבור אפליקציית דוחות בדק, כולל תמיכה גלובלית ב-RTL וממשק התחלתי בעברית.

## מה נכלל

- תשתית Expo עם TypeScript ו־strict mode.
- מבנה תיקיות ראשוני לשכבות מוצר (app/components/features/db/services/lib/docs).
- רכיבי UI בסיסיים בעברית: טקסט, קלט וכפתור.
- מעטפת ניווט בסיסית עם שלושה מסכים: פרויקטים, עורך פרויקט ולוח בקרה.

## החלטות טכניות

1. **RTL באתחול האפליקציה**
   - מופעל דרך `I18nManager.allowRTL(true)` ו-`forceRTL(true)` בעת האתחול.
   - רכיבי בסיס מגדירים `textAlign: 'right'` ו-`writingDirection: 'rtl'` כברירת מחדל.
2. **ניווט שלד ללא תלות חיצונית**
   - מצב ניווט לוקאלי באמצעות state כדי לאפשר תהליך הקמה מהיר.
   - ניתן להחליף בהמשך ל-React Navigation ללא שבירת שכבות UI.
3. **Baseline איכות קוד**
   - ESLint עבור TypeScript.
   - Prettier לאחידות פורמט.
4. **תאימות לכלי PR ללא קבצים בינאריים**
   - הוסרו אייקונים/תמונות PNG מה-repo כדי למנוע כשלים בכלי PR שלא תומכים בקבצים בינאריים.
   - ניתן להוסיף נכסי מותג בשלב מאוחר יותר דרך תהליך CI נפרד או אחסון חיצוני.

## צעדים הבאים

- מעבר לניווט production-grade.
- הוספת שכבת אחסון מקומית (למשל SQLite/Realm).
- חיבור API וזרימת אימות משתמש.
- הוספת נכסי אפליקציה (icon/splash/favicon) בתהליך שאינו חוסם יצירת PR.
